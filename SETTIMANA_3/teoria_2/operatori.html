OPERATORI

Un espressione è una combinazione di valori, variabili e operatori che rappresentano un nuovo valore.

Gli operatori determinano il risultato di un'espressione

Classifichiamo gli operatori in base al numero di valori che possono combinare: unari, binari, ternari

Operatori aritmetici

Operatori binari che combinano valori numerici.
+ è l'addizione
- è sottrazione
/ divisione
* moltiplicazione
% modulo o resto
esempi:
9/4
x +10
Seguono le regole di precedenza matematiche e come in matematica posso usare le parentesi ma solo le ()
4 + 5 * 6 + 7 = 41 (precedenza alle moltiplicazioni)
(4 + 5) * (6 + 7) = 117

Operatori aritmetici possono anche essere unari
++ incremento
-- decremento
- negazione -> x=5 -> -x = -5
incremento e decremento li posso applicare solo a delle variabili
-> è molto importante la posizione dell'operatore
-> se l'operatore è a sinistra (dell'operando) l'operazione viene eseguita prima di valutare l'espressione
-> se l'operatore è a destra (dell'operando) l'operazione viene eseguita dopo

Operatori logici

Consentono di combinare espressioni booleane (true) (false)
&& -> operatore binario -> and
|| -> operatore binario -> or
! -> operatore unario -> not
5 > 2 && 3 !=4 // true
!5 === 5 // false

= -> assegnato
== -> uguale
!= -> diverso
=== -> strettamente uguale
(uguale sia il valore che il tipo di dato)
-> es 0 e "0" non sono strettamente uguali (è uguale il valore ma non il tipo)

Operatore ? per assegnamenti condizionali

E' l'unico peratore ternario presente in javascript. Restituisce un valore in base a una espressione booleana

Sintassi: condizione ? valore1 : valore2
verifico se la condizione è vera (true|false) -> se è vera viene restituito il valore 1 altrimenti il valore 2

Operatori su stringhe

operatore concatenazione (+) -> crea una nuova stringa come risultato della concatenazione di due stringhe
