Oggetto console

L'oggetto console, insieme a editor e browser è fondamentale per scrivere codice javascript

E' uno degli strumenti fondamentali (insieme all'editor e al compilatore) per scrivere qualsiasi linguaggio di programmazione

Ci permette infatti di fare il debug, il controllo, del flusso del nostro codice

Inserire codice javascript in una pagina html

  • a) importare file con codice javascript esterno (consigliato)
  • b) scrivere blocchi di codice nella pagina
  • c) inserire codice inline

Spesso la console si usa per tenere traccia dell'evoluzione del programma o per verificare i valori

Inserire codice inline

Posso inserire direttamente le istruzioni javascript nel codice di un elemento html, assegnandolo ad un attributo che rappresenta un evento

Altro approccio per inserire codice inline, ma è valido solo per i link:

Inserisco blocchi di codice

Pro: risulta immediato quando dobbiamo mettere in relazione codice da eseguire con un elemento html

Contro: in caso di codice minimamente complesso (variabili, funzioni, console) non risulta comodo

tag head e tag body? se il codice js interagisce con un elemento HTML ... essere sicuri che l'elemento è già parsato
così l'elemento sarà già disponibile in memoria ... questo spiega perchè a volte troviamo gli script prima della chiusura di body

Inserisco javascript esterno

Approccio più consigliato. Consiste nel collegare una pagina HTML con uno script presente in un file esterno.
Il grande vantaggio è la separazione tra struttura del documento e codice, tra cos'è un tag e cosa deve fare

Aspetti sintatti fondamentali

Il codice javascript è una sequenza di istruzioni che vengono interpretate e eseguite

Ogni istruzione è delimitata da ;

inserire sempre la punteggiatura di terminazione per evitare ambiguità e risultati inattesi

2. Javascript è case sensitive cioè fa distinzione tra maiuscole e minuscole

ad esempio se assegno come nome di una variabile MiaVariabile e poi la richiamo con Miavariabile sono due cose diverse

3. I commenti

Javascript prevede la possibilità di inserire commenti a singola riga o multi riga

4. Gli spazi bianchi

Non assumono significati particolari, li posso inserire per aumentare la legibilità del codice

Eccezione: gli spazi all'interno delle stringhe (domani)
